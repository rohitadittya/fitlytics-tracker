# Fitlytics

Fitlytics is a comprehensive fitness tracking web application built with Vue 3 and TypeScript. It allows users to log their fitness activities, view statistics, and manage their profile.

## Features

- **User Authentication**: Secure login and registration system.
- **Activity Tracking**: Log various fitness activities including Running, Cycling, Swimming, Yoga, and Strength Training.
- **Activity Management**: Edit and delete your own activities.
- **Statistics Dashboard**: View detailed statistics and charts for your fitness activities.
- **Profile Management**: Update your personal information (age, gender, height, weight).
- **Admin Dashboard**: View all users and their activities (admin only).
- **Responsive Design**: Built with Bulma for a modern, responsive UI.

## Tech Stack

- **Framework**: [Vue 3](https://vuejs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **State Management**: [Pinia](https://pinia.vuejs.org/)
- **Routing**: [Vue Router](https://router.vuejs.org/)
- **Styling**: [Bulma](https://bulma.io/)
- **HTTP Client**: [Axios](https://axios-http.com/)

## Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/rohitadittya/fitlytics.git
   cd fitlytics
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```
## Usage

1. **Get Started**: Open the application locally and log in to your account.
2. **Dashboard Overview**: View all fitness activities in the main dashboard or dive into your personal stats page to see progress.
3. **Log & Manage Activities**: Navigate to the 'Activities' tab to track recent exercises like running or swimming, or delete old ones.
4. **Update Profile**: Keep your personal details up to date via the 'Profile' section.
5. **Administration**: If logged in as an admin provider, you can manage all users directly from the 'Admin' panel.
